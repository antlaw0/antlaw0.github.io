AI Cooking Game

The AI Cooking Game

This is a more recent addition to my collection. This is a full-stack browser-based game. It uses AI to make a text-only cooking simulator. Basically, it is like a Dungeon Master in a game of Dungeons and Dragons where you type what you want to do and the AI tells you the outcome. You start with nothing but a street-side food cart and a hotplate and cook breakfast, lunch, dinner, and dessert each day to earn money based off your cooking knowledge and technique. The AI gives you a score out of 10 which determines how much money you earn. Money is used at the Cooking Emporium to buy more ingredients and cooking supplies and eventually better cooking venues such as a food truck and restaurant. The key is how much description about what and how you cook that determines your success so the better cooks out there will obviously do better at this game. Did I mention it is completely screen reader accessible?

The Braille Game

The Braille Game

Following my totally original naming style, we have The Braille Game. This is a static web app that uses JavaScript to simulate keys on a Braille keyboard using the home row keys. The more letters you can type correctly in a minute, the higher score you get. The game uses the JavaScript's built-in text to speech API to speak the dots or letters that appear on the screen. I find the TTS API immensely useful for making accessible apps and games (link to MDN Web Docs for the JavaScript TTS API).
